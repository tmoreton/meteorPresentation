METEOR.js

What is Meteor.js? What makes it different

  • Meteor is an Open-source Fullstack Javascript Framework/Platform.
  • An application built with Meteor runs on both the client
  • Easily ported to Browser and Mobile Apps
  • Live updates to the database with web sockets
  • Unified package system with Atmosphere.js
  • One of the first frameworks to essentially encompass the entire MEAN stack
